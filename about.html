About MITAOE Library

This demo UI models a campus library system with student registration, login, book browsing, issue and return flows. Data is stored locally (localStorage) to demo functionality without a backend.

Rules (sample)

  • Issue period: 14 days
  • Late fee: ₹2/day
  • Max books per student: 3
