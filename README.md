# 📚 Library Management System

A responsive Library Management System developed using **HTML, CSS, JavaScript, and LocalStorage** to manage books, student records, and issuing operations.

🌐 **Live Demo:** https://mitaoe-library-management-system.netlify.app/

## ✨ Features

* Student Signup & Login
* Book Search and View
* Book Issue & Return
* Student Dashboard
* Staff Login & Book Management
* Data Storage using LocalStorage
* Responsive User Interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* LocalStorage
* VS Code

## 📖 Project Overview

The system provides separate functionalities for students and staff. Students can register, log in, view books, and manage issued books, while staff can manage the book inventory through an easy-to-use interface.
